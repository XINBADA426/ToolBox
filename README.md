# ToolBox
This is the script I used for bioinfo analysis

## fq
Scripts to deal with fq/fq.gz files

### fqcombain.py
合并分批下机的原始数据


## fa
Scripts to deal with fa files

### ORFfinder.py
寻找最长转录本


## table
Scripts to deal with tables

### formattable.py
以较为美观的方式打印表格

## igv
Scripts to use igv to plot pic

## one_gene_bam.py
使用基因序列及基因比对bam文件获取read在该基因的分布情况
